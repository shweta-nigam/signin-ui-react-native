# Sign In UI - React Native

A simple and clean Sign In screen built using React Native and Expo.

## Features

- Email and password input fields
- Password hide/show icon
- Social media sign in buttons
- Keyboard avoiding behavior
- Custom app logo
- Green themed design

## Tech Stack

- React Native
- Expo
- TypeScript
- Expo Vector Icons
